Deals
• Source origin = Marketplace → Website / Call Booking
• Source origin = ManuallyCreated + no Lead Created Date → Manually Added
• Source origin = ManuallyCreated + Lead Created Date set → Converted Lead
• Source origin = Chatbot → Chatbot
• Source origin = WebForms → Conference (Source channel ID = form name)
• Source origin = API → API / Integration

Leads (API active + archived — same logic)
• Source origin = Marketplace + Source channel ID = Zapier + utm_source = website (or blank) → Website Booking
• Source origin = Marketplace + Source channel ID = Zapier + utm_source = other → Personal Link
• Source origin = Chatbot + Source channel ID = [chatbot name] → Chatbot
• Source origin = Marketplace + Source channel ID = Surfe → Surfe / LinkedIn
• Source origin = WebForms + Source channel ID = [form name] → Conference
• Source origin = API → Converted from Deal
• Source origin = ManuallyCreated → Manually Added

Sync Archived Leads to Airtable

Export archived leads from Pipedrive: Reports → Leads → filter by Status = Archived → Export CSV.
This will overwrite all previous data in Airtable.
